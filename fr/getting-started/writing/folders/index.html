Dossiers

Configurer les nuds de rpertoire via les fichiers _dir.yml.

Vous pouvez personnaliser les dossiers en crant un fichier _dir.yml dans le dossier.

_dir.yml
Structure du rpertoire
title: Dossier
icon: lucide:folder
navigation.redirect: /frameworks/vue
navBadges:
  - value: Nouveau
    type: lime
sidebar:
  style: group
collapse: false

content/
  1.frameworks/
    _dir.yml
    1.vue.md
    2.nuxt.md

Paramtres

titlestring
Titre de la page. Par dfaut, utilise le nom du fichier.
iconstring
L'icne affiche dans la barre latrale pour la page.
navBadges[]
Une liste de badges ajouts la fin du lien de la barre latrale.
collapseboolean
Si le dossier doit tre rduit par dfaut.
sidebarobject
style'default' | 'tree' | 'group'
default
Le style utilis pour le dossier. La valeur par dfaut peut tre configure dans appConfig.
defaulttreegroup
