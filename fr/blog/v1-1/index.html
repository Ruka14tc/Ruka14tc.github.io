Zoom sur l'image

Cliquez sur l'image pour zoomer et dzoomer. Cette fonctionnalit est utile pour les images ncessitant une visualisation dtaille.

![Image](/prose-img/light.png)
Lire la suite Components > Prose Components > Images and embeds

Images selon le mode couleur

Les images changent en fonction du mode couleur actuel (clair ou sombre). Ceci est utile pour les images ayant des couleurs diffrentes selon le mode.

Cliquez sur le bouton de changement de mode pour voir l'image changer.

::color-mode-image
---
srcLight: /prose-img/light.png
srcDark: /prose-img/dark.png
alt: Image mode couleur
---
::
Lire la suite Components > Prose Components > Images and embeds

Aire de jeu

Vous pouvez utiliser le composant ::playground pour intgrer des exemples de code interactifs dans votre documentation. Il prend en charge les fournisseurs StackBlitz et CodeSandbox.

::playground
---
provider: stackblitz
repo: nuxt/starter
branch: v3
file: app.vue
---
::
Lire la suite Components > Docs Components > Playground
