Vue d'ensemble

Personnalisation de shadcn-docs-nuxt.

shadcn-docs-nuxt est configur via app.config.ts.

app.config.ts
export default defineAppConfig({
  shadcnDocs: {
    header: {
      // ...
    },
    aside: {
      // ...
    },
    main: {
      // ...
    },
    // ...
  },
});

Rfrence des Champs

Dclarations de types illustratives :

type Target = '_blank' | '_parent' | '_self' | '_top' | (string & object) | null | undefined;

interface ILink {
  title?: string;
  icon?: string;
  to: string;
  target?: Target;
  showLinkIcon?: boolean; // Pour la section des liens de navigation et de la table des matires
}

interface INav {
  title: string;
  links?: (ILink & { description: string; icon: string })[];
  to?: string;
  target?: Target;
}
