角色設置

角色設置

操作設定

通常

  • 操作類型 → 改為 傳統
    這和移動和轉向方式有關，可以直接按後退就改變角色面向
    對於躲避一些機制比較方面，轉向後再移動也會比較快速
  • 過場動畫設置 → 三個都勾選

  • 任務進行中對其他玩家顯示設定 → 初期勾選
    一開始會有很多人再一起找同一個NPC導致點不到NPC
    勾選後可以在靠近時暫時隱藏周圍玩家
  • 視角修正
    手把玩家的轉向速度可以稍微條快一點

目標

  • 設置目標
    建議先依照這個設定
  • 設置自動轉中目標
    建議都勾選上，尤其補師在補完血之後施放攻擊技能可以不用重新選目標

分類

  • 地面目標設定 也是兩個都都勾選上，主要對於需要放置地面技能(例如白魔法)

  • 拔出和收回武器設定
    應該也是屬於手把玩家設定?
    可以在左右切換目標時可以被選擇的對象

角色

  • 外觀設定
    顯示和調整頭部設定可以在角色的左下設定
    拿出武器時也會有兩種動作(戰鬥、放鬆)，這邊則是設定是否要切換與切換時間

戰鬥特效

覺得全部開太亂的話可以依照需求逐漸關閉，但PVP建議開啟來判斷敵方特效技能

  • 滑鼠設定
    可以選自己直接改狀態(離開、RP)和簽名內容

物品設定

物品欄顯示類型 → 改為 全展開
雇員物品欄顯示類型 → 改為 展開
獲得物品前後往前收納 → 勾選 比較容易區分新獲得物品

  • 商店設置 → 不勾選 初期東西大多沒什麼用途，可以直接賣掉

介面設定

  • 選單設定
    手把玩家建議勾選按紐型
    可以在沒用手把時也可以簡單操作選單

推薦任務和資訊中心可以關閉，應該也沒人會看吧...?

名牌設定

設定有點多，但主要就是HP減少才顯示、召喚獸和寵物點選才顯示
設定前方的圓形圖示則可以更改顯示顏色

  • 啟用職能顏色 → 勾選
    在副本內比較方便判斷坦補位置

快速鍵設置

顯示

由於我是手把玩家，快速鍵3~10都拿去做聚集了，會另外再巨集做說明

  • 倒數計時位置 → 圖示中央
  • 不顯示未設定格子 → 勾選
    不會顯示那個框框而已，如果要拖曳技能上去還是可以的

共用

一個角色分別各有10(鍵盤)/8(手把)組快速鍵
可以把一些設定成共用，就不會切換職業之後就要重新設定

訊息視窗設定

  • 聊天輸入框
    字型大小可以依照螢幕條大，2K螢幕14~16為佳

  • 訊息設置
    FF14最多只能設定4組聊天欄(預設3組)

訊息詳情

聊天室窗字體大小也是在這邊設定
需要注意是分開設定，所以要調就全部一起條大，也可以勾選顯示訊息時間

訊息提醒

多勾選感情動作，有人對自己做動作可以有通知
